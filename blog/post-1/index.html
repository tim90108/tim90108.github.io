Canvas 基礎教學

By 邱謝仁欽 03 April 2025

🎨 Canvas 是什麼？

<canvas> 是 HTML5 提供的一個元素，讓你可以在網頁上動態繪製圖形、圖片處理、動畫，甚至是遊戲開發。你可以使用 JavaScript 操控它的繪圖上下文（context），來畫出各種 2D 或 3D 圖形。

✏️ 如何使用 Canvas？

1. HTML 結構

<canvas id="myCanvas" width="400" height="300" style="border:1px solid #ccc;"></canvas>
  • id：指定一個識別名稱，供 JavaScript 使用。
  • width / height：設定繪圖區域大小。
  • style：加個邊框比較看得清楚。

2. JavaScript 畫圖

const canvas = document.getElementById('myCanvas');
const ctx = canvas.getContext('2d'); // 取得 2D 繪圖上下文

// 畫一個紅色矩形
ctx.fillStyle = 'red';
ctx.fillRect(50, 50, 150, 100); // (x, y, width, height)

3. 更多繪圖功能

畫圓形

ctx.beginPath();
ctx.arc(200, 150, 40, 0, Math.PI * 2); // x, y, radius, startAngle, endAngle
ctx.fillStyle = 'blue';
ctx.fill();

畫線條

ctx.beginPath();
ctx.moveTo(20, 20);
ctx.lineTo(200, 100);
ctx.strokeStyle = 'green';
ctx.lineWidth = 3;
ctx.stroke();

🚀 小動畫範例：移動的圓形

<canvas id="canvasMove" width="400" height="200" style="border:1px solid #ccc;"></canvas>
<script>
  const canvas = document.getElementById('canvasMove');
  const ctx = canvas.getContext('2d');
  let x = 20;

  function draw() {
    ctx.clearRect(0, 0, canvas.width, canvas.height); // 清除畫面
    ctx.beginPath();
    ctx.arc(x, 100, 20, 0, Math.PI * 2);
    ctx.fillStyle = 'purple';
    ctx.fill();
    x += 2;
    if (x > canvas.width) x = 0;
    requestAnimationFrame(draw); // 每幀呼叫
  }

  draw();
</script>

📚 小結

Canvas 是一個功能強大的工具，能讓你：

  • 動態產生圖形
  • 實現動畫效果
  • 建立遊戲或視覺化資料展示

只需結合 JavaScript，就能創造無限可能！
